# ⚽ ScoutStats

ScoutStats is a football performance tracking platform that allows players to document matches, monitor progress, and reflect on their development over time.

Instead of storing match performances in notes apps or spreadsheets, players can build a digital football journal with statistics, ratings, match reflections, and performance comparisons.

---

## 🚀 Features

### 👤 Player Profiles

Create a football profile including:

- Name
- Age
- Position
- Team

---

### ⚽ Match Logging

Record details from every match:

- Opponent
- Competition
- Match Date
- Result
- Minutes Played
- Goals
- Assists
- Shots On Target
- Shots Off Target
- 1v1 Won
- 1v1 Lost
- Match Reflection

---

### ⭐ Automatic Performance Ratings

ScoutStats automatically calculates a match rating based on player performance.

Ratings consider:

- Goals
- Assists
- Shots On Target
- Successful 1v1s
- Missed Chances
- Lost Duels
- Match Result

---

### 📖 Football Journal

Every match is stored in a personal football journal, allowing players to review previous performances and track improvement over time.

---

### 📊 Dashboard Analytics

The dashboard displays:

- Matches Played
- Total Goals
- Total Assists
- Average Rating
- Best Rating
- Personal Records
- Recent Form

---

### 📈 Performance Tracking

Players can monitor their football journey through:

- Match history
- Average ratings
- Performance trends
- Statistical summaries

---

### 📊 Comparison Centre

Players can compare their statistics against benchmark values for their position.

Examples:

- Goals Per Match
- Assists Per Match
- Average Rating

This provides quick insight into whether a player is performing above or below expected standards.

---
