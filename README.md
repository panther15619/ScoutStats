# ⚽ ScoutStats

ScoutStats is a football performance tracking platform that allows players to document matches, monitor progress, and analyse their development over time.

The platform acts as a digital football journal where players can record performances, track statistics, receive automatic ratings, and compare their performance against position-based benchmarks.

---

## 🚀 Features

### 👤 Player Profiles

Create a personalised football profile including:

- Full Name
- Age
- Position
- Team

---

### ⚽ Match Logging

Record key information from every match:

- Opponent
- Competition
- Match Date
- Match Result
- Minutes Played
- Goals
- Assists
- Shots On Target
- Shots Off Target
- 1v1 Won
- 1v1 Lost
- Match Reflection
